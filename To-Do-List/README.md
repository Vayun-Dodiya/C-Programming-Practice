# To-Do List

## Description
A lightweight task manager focused on efficient in-memory array manipulation and terminal user interface design.

## Key Features
* **Dynamic Shifting**: Efficiently handles task deletion by shifting array elements to fill gaps, maintaining a continuous and ordered list.
* **Input Protection**: Implements buffer clearing and specific `scanf` formatting to prevent overflows and handle multi-word task entries.
* **UI Persistence**: Features a clean interface with automatic screen clearing to provide a smooth user experience.

## Technical Skills
* **Array Manipulation**: Mastery of multidimensional `char` arrays for string storage.
* **String Library**: Extensive use of `strcpy()` and `strlen()` for list management.
* **Cross-Platform Compatibility**: Uses preprocessor directives (`#ifdef _WIN32`) for portable screen clearing logic.

## Highlights
The project showcases a strong grasp of foundational C logic, specifically in managing the lifecycle of data within an array structure and preventing common input errors.
